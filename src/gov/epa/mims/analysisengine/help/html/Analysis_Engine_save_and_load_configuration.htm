Save Configuration

 

The Analysis Engine allows you to save configuration settings so that they can be applied to different data sets with the same structure.  The configuration settings always include the settings of the Filter Rows function, the Format Columns function, and the Sort Columns function.  Plot configurations are also listed for every plot that has been created from the front table.   When you choose the File/Save Configuration menu item in the Data Table Window or click on the Analysis Configuration () toolbar button, the Analysis Configuration window appears:

If you double-click on a name in the Configurations column, the configuration name can be altered.  You may also click on the boxes in the "Save?" column to indicate whether or not each configuration should be saved.  Enter a file path and file name in the Save As field in order to save the configuration settings. or click Browse to choose the file path.

The Delete, Select, and Clear buttons offer functions to help you decide which configurations should be saved.   Click Delete to permanently remove the selected plot configurations from the list.  Click Select to check the Save? check boxes beside all highlighted configurations. Click Clear to un-check the Save? check boxes beside all highlighted configurations.

The View and Edit buttons let you re-create the plots that have been previously produced.  If you highlight one of the plot configurations and click the View button, the pdf file displaying that plot is created immediately.  If you highlight one of the plot configurations and click the Edit button, the plotting engine interface is produced and shows all of the previously set parameters.

After choosing the appropriate configurations and specifying a file name in the Save As field, click OK to store the configuration settings.  

Press Cancel to close the dialog box without saving any changes.
