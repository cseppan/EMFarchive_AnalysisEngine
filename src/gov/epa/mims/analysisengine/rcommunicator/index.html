Package: gov.epa.mims.analysisengine.rcommunicator

This package automates the generation of statistical plots using R-plot and the gov.epa.mims.analysisengine.tree package. A description on how to build composite trees describing plots to automate can be found at gov.epa.mims.analysisengine.tree package .
 
 
 
CURRENTLY IMPLEMENTED PLOTS

user selected bar spacing and category spacing
stackable bars
horizontal bars
