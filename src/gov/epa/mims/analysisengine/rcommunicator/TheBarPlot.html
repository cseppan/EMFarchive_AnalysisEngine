The BarPlot

Overview

The BarPlot displays a 2-dimensional plot of categorical data. A category is represented by a DoubleSeries data object which contains a value for each element in the category. A category can be displayed as individual bars within a category or as a single stacked bar. The user can control the color and width of the bars as well as the separation between individual bars and the separation between categories. The bars can be drawn horizontally or vertically. The axes are fully configurable with settings for color; tickmark locations; tickmark label font, color, size and orientation; and total control over axis labels.

Adding Data

The DoubleSeries.java object is initialized with a series of double values and then added to the DataSets tree component. The user associates a "key" java.lang.Object with the DoubleSeries. The "key" and the DoubleSeries objects are added to the DataSets tree component. The "key" is then added to the BarPlot component.
 

example code

 
 
 

// allocate objects
ds1 = new DoubleSeries();
ds2 = new DoubleSeries();
ds3 = new DoubleSeries();
ds4 = new DoubleSeries();
// initialize data
ds1.setData( new double[] {  0.41749691, 0.56145016, 1.57874680 });
ds2.setData( "initialization has been elided" );
ds3.setData( "initialization has been elided" );
ds4.setData( "initialization has been elided" );
// name data sets
ds1.setName("My data set 1");
ds2.setName("My data set 2");
ds3.setName("My data set 3");
ds4.setName("My data set 4");
// create unique Object keys
Object key1 = new Integer(1);
Object key2 = new Integer(2);
Object key3 = new Integer(3);
Object key4 = new Integer(4);
//add data to DataSets
dataSets.add(ds1, key1);
dataSets.add(ds2, key2);
dataSets.add(ds3, key3);
dataSets.add(ds4, key4);
// add data keys to Plot
barPlot.addKey(key2);
barPlot.addKey(key3);
barPlot.addKey(key4);

 

Adding Analysis Options

Analysis options are classes with methods for setting different display parameters. The AnalysisOptions component is not required to generate a plot; however, it is highly recommended since it is how user preferences are set.
 

example code (controlling Bar data display)


//create and initialize a BarType analysis option
barType = new BarType();
barType.setColor( new Color[]{ Color.blue ,Color.red ,Color.green ,Color.yellow ,} ); 
barType.setBorderColor( Color.green );
barType.setHorizontal( false );
barType.setStacked( false );
barType.setSpaceBetweenBars( 1.2 );
barType.setSpaceBetweenCategories( 2.0 );
barType.setWidth( new double[] { 0.5 } );
// add BarType to AnalysisOptions
analysisOptions.addOption( "BAR_TYPE" , barType );
