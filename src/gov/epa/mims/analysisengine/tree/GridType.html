GridType
METHOD DESCRIPTION
setColor(java.awt.Color) color of grid lines
setLineStyle(java.lang.String) "SOLID","DASHED","DOTTED", "DOTDASH","LONGDASH","TWODASH",
"BLANK"=no plotting, Alternatively, a string of up to 8 characters (from 0:9, "A":"F") may be given, giving the length of line segments which are alternatively drawn and skipped. Used cyclically.
setLineWidth(double) width of grid line
setNumberXcells(int) Number of cells of the grid in x direction.  "-1" causes the grid to default to the number of ticks marks on the corresponding axis.
setNumberYcells(int) Number of cells of the grid in y direction.  "-1" causes the grid to default to the number of ticks marks on the corresponding axis.
setDraw(boolean) true = draw the grid; false = do not draw grid
