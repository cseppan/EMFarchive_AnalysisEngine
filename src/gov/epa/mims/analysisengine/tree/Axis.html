Axis
METHOD DESCRIPTION
setAxisLabelText(gov.epa.mims.analysisengine.tree.Text) add Text object describing the axis label
setDrawOpposingAxis(boolean) true = draw opposing axis; default = false
setDrawAxis(boolean) true = draw this axis; false= no axis is drawn
setDrawTickMarks(boolean) true = draw tick marks
setDrawTickMarkLabels(boolean) true = label the tick marks
setDrawTickMarkLabelsPerpendicularToAxis(boolean) true = labels are drawn perpendicular to the axis; default = false
setLinesIntoMargin(double) number of lines to shift position of axis
setPosition(double) absolute position of axis in user coordinates
setAxisColor(java.awt.Color) axis color
setTickMarkLabelColor(java.awt.Color) label color
setTickMarkFont(java.lang.String) font for tick mark labels; "plain","bold","italic","bold italic"
setTickMarkLabelExpansion(double) expansion coefficient for tick mark labels
setUserTickMarkPositions(double[]) user selected tickmark positions; overrides default positions
setUserTickMarkLabels(java.lang.String[]) user selected tickmark labels; overrides default labels
clone() clones this object
