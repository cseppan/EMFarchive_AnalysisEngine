OutlineType
METHOD DESCRIPTION
setColor(java.awt.Color) line color
setLineStyle(java.lang.String) "SOLID","DASHED","DOTTED", "DOTDASH","LONGDASH","TWODASH", "BLANK"=no plotting, Alternatively, a string of up to 8 characters (from 0:9, "A"
:"F") may be given, giving the length of line segments which are alternatively drawn and skipped.
setLineWidth(double) Width of lines.
setDraw(boolean) true = draw grid ; false = do not draw grid
